如你所见，这是一个Material风格的Hexo主题

主题预览：https://blog.lemonneko.moe

### 安装
很简单呀，直接克隆就好
```bash
cd path/to/your/blog
git clone https://github.com/LemonNekoGH/material-theme-for-hexo themes/material-theme-for-hexo
```
然后去_config.yml设置一下
```yaml
theme: material-theme-for-hexo
```
